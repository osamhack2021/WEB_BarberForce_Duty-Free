# Backend 개발일지
___
## DB
- 사용DB : MongoDB
- DB 데이터  
  + 사용자 정보
    1. 이름
    2. 이메일(ID)
    3. PASSWORD
    4. 군번
    5. 본인 확인 정보 ex)휴대폰 번호, 주민등록번호
    6. 소속 부대(대략적인 위치)
    7. 회원 유형(장병, 제휴 미용실)
  + 미욜실 정보
    1. 미용실 위치
    2. 소개 사진
    3. 소개글
    4. 예약 가능 시간
  + 예약 정보(현황)
  + 리뷰
